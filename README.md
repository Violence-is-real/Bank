# Описание проекта

В рамках практической работы был разработан консольный проект **Bank**, содержащий класс `BankAccount`.

Данный класс моделирует работу банковского счета и позволяет выполнять следующие операции:

* хранение имени владельца счета
* хранение текущего баланса
* списание средств со счета (`Debit`)
* пополнение счета (`Credit`)
* обработку исключений при некорректных операциях

Для проверки корректности работы методов был создан отдельный проект модульного тестирования **BankTests**.

# Вывод

В ходе выполнения практической работы было изучено использование модульного тестирования при разработке программного обеспечения.

Был создан консольный проект **Bank**, содержащий класс `BankAccount`, реализующий операции работы с банковским счетом.

Для проверки корректности работы программы был разработан отдельный проект модульных тестов **BankTests**. С помощью тестов была проверена корректность работы методов `Debit` и `Credit`, а также обработка исключительных ситуаций.

Использование автоматизированного тестирования позволяет повысить надежность программного кода и значительно упростить поиск и исправление ошибок в процессе разработки.
